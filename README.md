# ARC V13 正式版｜統計表格、中文搜尋與基礎資料設定修正版

本版針對第 20 次需求調整：

- 各項目申請數據｜本月 / 本年恢復為統計表格，不改成卡片。
- 年度累計紀錄恢復為統計表格，不改成卡片。
- 統計表格移除內層滑動，透過欄寬、padding 與自動換行處理顯示。
- 全系統搜尋 / 查詢 / 篩選輸入加強中文注音 composition 處理與 debounce。
- 搜尋輸入狀態與實際查詢關鍵字分離，避免組字中途重新渲染、失焦或固定文字。
- 系統設定移除「總覽」模式，只保留管理員可新增、修改、停用或刪除的可維護項目。
- 系統設定新增 / 整理：申請項目、仲介、帳戶、承辦 / 人員 / 帳號、傳真 / 領件、提醒事項、列印設定。
- 行政不可調整帳戶餘額；餘額調整維持管理員與財務角色可操作。
- 系統設定與帳戶餘額異動會寫入操作紀錄。

此版延續 V13 樣式、正式版 Supabase 架構與既有資料儲存邏輯。
