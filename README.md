# ARC V13 正式版｜統計恢復＋傳真搜尋修正＋領件日與收據順序排序

本版依第十五項需求調整，維持既有 V13 內容、呈現方式、樣式、權限、Supabase app_state 儲存邏輯與 icon。

## 本次更新

- 恢復「每月每個人申請件數統計」
  - 依承辦 / 行政人員分組
  - 顯示年月、承辦 / 行政人員、各申請項目件數、申請件數、合計件數
- 新增 / 恢復「各項目申請數據」
  - 本月各申請項目件數
  - 本年各申請項目件數
  - 每月趨勢紀錄
  - 年度累計紀錄
- 修正【移民署傳真領件】搜尋欄位中文輸入卡住問題
  - 支援注音中文組字
  - compositionstart / compositionend 處理
  - debounce 延遲搜尋
- 預計領件區「領件日」預設為下一週週四
  - 今日若為週四，也會預設下一週週四
- 收據順序不可重複
  - 同一領件日 / 同一批預計領件資料不可重複
  - 重複時不可加入預計領件區、不可列印、不可建立傳真領件紀錄
- 移民署傳真領件資料顯示與列印排序一致
  - 第一關：收費日期由早到晚
  - 第二關：收件編號遞增
  - 第三關：外字五碼遞增

## 部署

本次為前端功能與 app_state JSON 使用邏輯調整，不需要重新跑 Supabase SQL。
