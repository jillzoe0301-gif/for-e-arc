# ARC V13 正式版｜資料異動登記＋批次垂直貼上＋帳戶餘額分開顯示

本版依第十六項需求調整，維持既有 V13 內容、呈現方式、樣式、權限、Supabase app_state 儲存邏輯與 icon。

## 本次更新

- 修正「資料異動」案件可正常登記
  - 支援單筆案件登記
  - 支援批次送件
  - 金額為 0 或特殊金額時不會被系統直接阻擋
  - 可進入繳費、財務查詢、財務對帳、統計、傳真/領件與列印流程
- 批次送件支援垂直貼上
  - 申請項目可從 Excel / Google Sheet 垂直貼上
  - 金額可從 Excel / Google Sheet 垂直貼上
  - 貼上列數超過目前列數會自動新增列
  - 金額會自動清除逗號、空白與符號，例如 1,000、$1,000 會轉成 1000
- 操作欄位移除「上一列」按鈕
  - 保留刪除列、新增列、儲存等既有操作
- 灃康不同扣款帳號餘額分開顯示
  - 不再只顯示灃康合計餘額
  - 會依帳號 / 銀行 / 末五碼 / 備註個別顯示
  - 繳費、扣款、餘額更新、財務查詢及對帳仍依實際選擇帳戶分開計算

## 部署

本次為前端功能與 app_state JSON 使用邏輯調整，不需要重新跑 Supabase SQL。
