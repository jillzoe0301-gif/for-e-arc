# ARC V13 正式版｜統計欄寬＋全系統搜尋輸入＋收據順序防重修正版

本版修正第十七項需求：

- 各項目申請數據「申請項目」欄位縮窄，但仍保留完整可讀文字。
- 年度累計紀錄「年度」欄位縮窄，只保留顯示年度所需寬度。
- 首頁「最新待處理案件」文字加大，繳款與領件狀態更清楚。
- 全系統主要搜尋欄位改為支援中文注音 composition 與 debounce，避免輸入一個字就跳掉或失焦。
- 搜尋欄位更新結果後會自動恢復焦點。
- 移民署傳真領件「收據順序」不可重複。
- 收據順序重複時會提示目前已使用最大號碼與建議下一個可用號碼。
- 收據順序重複時不可加入預計領件區、不可列印、不可建立傳真領件紀錄。
- 收據順序維持數字排序，不會出現 1、10、2 的錯誤排序。

此版不需要重新執行 Supabase SQL。
