# 部署步驟

## 1. Supabase

1. 建立 Supabase 專案。
2. SQL Editor 執行 `supabase/001_schema.sql`。
3. SQL Editor 執行 `supabase/002_seed_data.sql`。
4. 複製 Project URL 與 anon public key。

## 2. GitHub

```bash
git init
git add .
git commit -m "ARC residence control supabase v1"
git branch -M main
git remote add origin <你的 GitHub repo URL>
git push -u origin main
```

## 3. Vercel

1. Import GitHub Repository。
2. Framework 選 Vite。
3. Build Command：`npm run build`。
4. Output Directory：`dist`。
5. Environment Variables 加上：
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
6. Deploy。

## 4. 第一次使用

1. 打開 Vercel 網址。
2. 選擇本機使用者。
3. 系統鎖定該台電腦使用者。
4. 若要更換使用者，到「系統設定」輸入管理員解除碼。

## 5. 正式使用前檢查

- 到「仲介與扣款帳號」確認扣款帳號與初始餘額。
- 到「系統設定」修改管理員解除碼。
- 到「系統設定」確認桃園移民署電話、傳真、傳真下方公司資料。
- 到「系統設定」確認承辦與申請項目。
