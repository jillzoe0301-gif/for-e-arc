# 部署步驟

1. 解壓縮本 ZIP。
2. 將資料夾內所有檔案放到 GitHub 專案第一層。
3. 確認第一層有：
   - index.html
   - package.json
   - package-lock.json
   - README.md
   - 部署步驟.md
   - .env.example
   - supabase/
4. 執行：

```bash
git add .
git commit -m "fix ARC V13 Chinese search and settings management"
git push origin main
```

5. Vercel 會自動重新部署。

本次不用重新跑 Supabase SQL。
