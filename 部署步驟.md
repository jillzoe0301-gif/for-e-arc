# 部署步驟

## 1. 解壓縮

請先解壓縮本 ZIP，不要直接把 ZIP 上傳到 GitHub。

GitHub 專案第一層需看到：

```text
index.html
package.json
package-lock.json
README.md
部署步驟.md
.env.example
supabase/
```

## 2. 推送到 GitHub

```bash
git add .
git commit -m "fix ARC V13 search full name and settings page"
git push origin main
```

## 3. Vercel

Vercel 會自動重新部署。

## 4. Supabase

這次不用重新跑 Supabase SQL。

若尚未完成正式版 SQL，才需要先執行：

```text
supabase/001_formal_schema_rpc.sql
supabase/002_fix_pgcrypto_login_and_hide_login_hint.sql
```
