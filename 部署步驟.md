# 部署步驟

## 1. 解壓縮

不要直接把 ZIP 放進 GitHub。請先解壓縮，並將資料夾內檔案放到 GitHub 專案第一層。

第一層應看到：

```text
index.html
package.json
package-lock.json
README.md
部署步驟.md
.env.example
supabase/
```

## 2. 推送 GitHub

```bash
git add .
git commit -m "add ARC V13 reentry permit direct case query flow"
git push origin main
```

## 3. Vercel

Vercel 會自動部署。

若需要手動重跑：

```text
Vercel → Deployments → Redeploy
```

## 4. Supabase

本次不需要重新執行 Supabase SQL。
