# ARC V13 正式版｜統計不滑動＋搜尋輸入＋系統設定＋權限＋傳真提醒修正版

本版修正第十八、十九項需求，維持 V13 內容、呈現方式、樣式與 icon 基準。

## 本版修正

- 【各項目申請數據｜本月 / 本年】改為卡片式直接顯示，不再使用內層滑動。
- 【年度累計紀錄】改為直接顯示卡片，不需左右或上下滑動。
- 全系統主要關鍵字搜尋欄位強化中文注音 composition 處理。
- 搜尋欄位加入 debounce 與搜尋暫存值，降低輸入中重渲染造成跳字、失焦或固定文字的狀況。
- 【系統設定】新增「系統設定總覽」，列出基本資料、居留案件、傳真/領件、財務、統計、權限、畫面顯示等可調整條件。
- 系統設定總覽顯示：設定名稱、目前設定值、可修改欄位、修改權限、最後修改人、最後修改時間。
- 行政角色不可調整帳戶餘額；餘額調整僅管理員與財務角色可操作。
- 帳戶餘額異動需輸入異動原因，操作紀錄保留異動前餘額、異動後餘額、異動金額、異動人與時間。
- 【傳真/領件】頁面移民署傳真領件區右上角新增紅字提醒：收據順序請寫在領件單右上角!
- 保留第十七項已完成的收據順序防重、統計欄寬、最新待處理案件字級與排序規則。

不需要。
