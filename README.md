# ARC V13 正式版｜會計批次對帳＋傳真領件補充版

本版維持 V13 內容、呈現方式、樣式與 icon 基準，針對本次指定功能調整：

## 本次更新

### 會計對帳確認
- 會計對帳與會計查詢可點選批次進入「對帳明細」。
- 批次明細上方新增「對帳完成」與「項目金額錯誤」。
- 對帳完成會記錄對帳完成時間、確認人、批次編號與狀態。
- 項目金額錯誤模式可針對單筆明細按「修正」。
- 單筆修正可修改申請項目、項目金額、備註 / 錯誤原因。
- 修正紀錄保留原申請項目、原金額、修正後資料、原因、修正人與時間。
- 批次明細左上角以醒目樣式顯示繳費日期、繳款人、仲介、帳戶名稱。

### 傳真與領件
- 「單筆領件」按鈕改到移民署傳真領件明細每筆資料最後方。
- 移民署傳真領件明細欄位順序已調整，並包含「舊卡」。
- 傳真領件單列印欄位已加入「舊卡」並依指定順序顯示。
- 預計領件區不再顯示單筆領件按鈕，只保留移除與領件日修改。
- 傳真領件紀錄新增刪除功能，僅管理員可操作。
- 刪除傳真領件紀錄前會確認，且不會刪除原始案件資料。
- 刪除傳真領件紀錄會留下操作紀錄。

## 套用方式

解壓縮後，將檔案放到 GitHub 專案第一層，再推送部署。

```bash
git add .
git commit -m "update ARC V13 accounting reconciliation and fax pickup controls"
git push origin main
```

本次為前端功能與 app_state JSON 結構延伸，不需要重新執行 Supabase SQL。
