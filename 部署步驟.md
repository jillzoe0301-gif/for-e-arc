# 部署步驟

## GitHub 上傳

不要直接上傳 ZIP。請先解壓縮，將以下檔案放到 GitHub 專案第一層：

```text
index.html
package.json
README.md
部署步驟.md
```

## 推送指令

```bash
git add .
git commit -m "rebuild ARC V13 with audit log and restore"
git push origin main
```

## Vercel 設定

```text
Framework Preset：Vite
Root Directory：./
Build Command：npm run build
Output Directory：dist
Install Command：npm install --no-audit --no-fund --legacy-peer-deps
Development Command：npm run dev
```

若使用 pnpm：

```text
Build Command：pnpm run build
Install Command：pnpm install --frozen-lockfile=false
Development Command：pnpm run dev
```

## 新增功能

- 刪除案件：到「案件查詢」點刪除，資料會進入「操作紀錄 / 資料救回」。
- 刪除繳費批次：繳費批次卡片右側點刪除批次。
- 刪除扣款帳號：仲介與扣款帳號頁刪除。
- 刪除行政選項：系統設定頁刪除。
- 救回資料：到「操作紀錄 / 資料救回」頁，點「救回」。
