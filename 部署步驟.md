# 部署步驟

## 1. 解壓縮

請先解壓縮本 ZIP，不要直接把 ZIP 上傳 GitHub。

GitHub 專案第一層需看到：

```text
index.html
package.json
package-lock.json
README.md
部署步驟.md
.env.example
supabase/
```

## 2. 上傳 GitHub

```bash
git add .
git commit -m "update ARC V13 fax pickup single receive print format and accounting search"
git push origin main
```

## 3. Vercel 自動部署

Vercel 設定維持：

```text
Framework Preset：Vite
Build Command：npm run build
Output Directory：dist
Install Command：npm install --no-audit --no-fund --legacy-peer-deps
```

## 4. Supabase

本次不用重新跑 Supabase SQL。
