# ARC V13 正式版｜提醒事項時間樣式與選單文字修正版

本版針對第十四項需求調整：

- 提醒事項「今天」判斷改為台北時間 Asia/Taipei。
- 星期一僅週一繳費、星期二僅週二傳真、星期四僅週四領件會顯示今天。
- 非當天提醒不會套用 #F4AE52 外框。
- 繳費提醒「乾坤、灃禾繳費前請先與財務確認。」字級加大並醒目顯示。
- 其他提醒事項文字加大。
- 財務對帳確認明細內「項目金額錯誤」按鈕改為 #DA4848 紅色白字。
- 批次對帳資訊內容文字改為 #30364F。
- 傳真領件列印單「收據順序」欄位字級縮小，只影響列印。
- 左側選單文字更新為：居留案件登記、居留證繳費、財務對帳確認、財務查詢、傳真/領件。

本次只修改前端顯示與樣式，不需重新執行 Supabase SQL。
