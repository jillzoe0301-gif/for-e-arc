# ARC V13 正式版｜傳真搜尋與提醒醒目顯示版

本版維持 V13 內容、呈現方式、樣式與 icon 基準，只針對本次指定的【傳真領件】搜尋與提醒事項顯示調整。

## 本次更新

### 移民署傳真領件搜尋
- 在【移民署傳真領件】區塊上方新增「雇主 / 工人 / 團號搜尋」。
- 可輸入雇主名稱、工人姓名、團號或部分關鍵字搜尋。
- 搜尋會同時比對雇主、工人、團號。
- 查無資料時顯示「查無符合資料」。
- 搜尋結果保留原本欄位、排序、勾選、列印、單筆領件功能。
- 一鍵全選 / 勾選待領件會以目前搜尋結果為範圍，不會誤選隱藏資料。

### 傳真領件提醒事項
- 傳真領件頁面新增提醒事項區，保留週一繳費、週二傳真、週四領件。
- 當天符合提醒事項時，該提醒會自動移到最前方。
- 當天提醒顯示「今天」標籤。
- 當天提醒外框使用 #F4AE52，背景使用 #FFF8EE。
- 非當天提醒維持原本樣式，不會誤套用橘色外框。

## 套用方式

解壓縮後，將檔案放到 GitHub 專案第一層，再推送部署。

```bash
git add .
git commit -m "update ARC V13 fax search and reminder highlight"
git push origin main
```

本次只調整前端顯示與搜尋，不需要重新執行 Supabase SQL。
