# 部署步驟

## 1. 解壓縮

不要直接把 ZIP 放進 GitHub。請先解壓縮，再把資料夾內的檔案放到 GitHub 專案第一層。

GitHub 第一層應該看到：

```text
index.html
package.json
package-lock.json
README.md
部署步驟.md
```

## 2. 推上 GitHub

```bash
git add .
git commit -m "rebuild ARC V13 with email login permissions audit restore"
git push origin main
```

## 3. Vercel 設定

建議使用：

```text
Framework Preset：Vite
Root Directory：./
Build Command：npm run build
Output Directory：dist
Install Command：npm install --no-audit --no-fund --legacy-peer-deps
Development Command：npm run dev
```

若使用 pnpm：

```text
Build Command：pnpm run build
Install Command：pnpm install --frozen-lockfile=false
Development Command：pnpm run dev
```

## 4. 預設帳號

```text
管理員：admin@forward.local / admin123
行政：arc@forward.local / arc123
會計：accounting@forward.local / acc123
```

## 5. 權限

```text
管理員：全部權限
行政：不可操作仲介與扣款帳號、系統設定、會計對帳、會計查詢
會計：只有會計對帳、會計查詢
```

## 6. 正式上線提醒

目前為 V13 單檔樣式版，帳號與資料存在瀏覽器 localStorage。若需要多人跨電腦共用同一份帳號、資料與操作紀錄，下一階段需改接 Supabase Auth + Supabase Database。
