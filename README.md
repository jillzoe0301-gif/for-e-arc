# ARC V13 正式版｜各項申請數據完整欄位＋金額 500 級距微調版

本版本延續 ARC V13 正式版 Supabase 架構與 V13 版面樣式，新增本次需求：

## 本版更新

- 統計數據頁「各項申請數據」改為完整欄位顯示。
- 各項申請數據會列出完整申請項目，即使該項目目前為 0 件也會顯示。
- 各項申請數據同時顯示：
  - 本月件數
  - 本月金額
  - 本年件數
  - 本年金額
  - 本月取消
  - 本年取消
- 取消案件不計入申請金額或繳款金額，改列於取消欄位與取消案件區。
- 金額欄位不再使用下拉選項。
- 金額欄位改為數字欄位，可用欄位右側上下箭頭依 500 級距往上或往下調整。
- 單筆送件、批次送件、選案繳費明細金額皆套用 500 step。
- 手動輸入非 500 倍數時，系統仍會整理成 500 級距。
- Excel 垂直貼上金額功能保留。

## 金額欄位規則

金額欄位為數字欄位：

- step：500
- min：0
- 可用上下箭頭調整：500、1000、1500、2000...
- 可手動輸入
- 可從 Excel 垂直貼上整欄金額

## 部署

解壓縮後，將檔案放到 GitHub 專案第一層，再推送到 main。

```bash
git add .
git commit -m "update ARC V13 stats item table and amount step input"
git push origin main
```

本次為前端功能與畫面調整，不需要重新執行 Supabase SQL。
